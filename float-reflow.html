html reflow

HTML reflow is a rendering technique that involves changes that affect the layout of a portion of the page. Reflow of elements cause the reflow that follows the child, the ancestor, or elements following it in the document object model.

To minimize reflow the user must also consider the document object model compatibilty. As Addy Osmani from Smashing Magazine puts it, speed is everything.
